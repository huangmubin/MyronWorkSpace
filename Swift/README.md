## AutoLayout

纯代码 AutoLayout 工具类

* 使用前须知
	1. superview: 父视图，约束需要添加到它身上
	2. first: 第一计算视图，用来添加约束的子视图，对应 firstItem
	3. second: 第二计算子视图，对应 secondItem，假如不设置，默认是 superview
	4. 约束计算公式： first = second * multiplier + constant
